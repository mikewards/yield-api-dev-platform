Get started with the prototype

1

Create your account

Sign up for free — takes 30 seconds.

2

Get your API key

Create an application in the Dashboard, then generate an API key.

  • Go to Dashboard → click Applications
  • Select + Create Application and name the App
  • From your app → click Create Key
  • Select Sandbox environment
  • Copy your API key
3

Make your first request

Run this command in your terminal. Replace YOUR_API_KEY with your key.

Terminal 
curl https://flow-platform-flow-platform-staging.up.railway.app/v1/yield/rates \
  -H "Authorization: Bearer YOUR_API_KEY"
