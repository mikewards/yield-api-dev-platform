API Reference

Complete API documentation for TBD's DeFi yield platform. Our REST APIs wrap Morpho and Aave protocols for a unified interface.

Dev Essentials

Authenticate your app, manage accounts, create access tokens, and subscribe to webhooks.

Authentication

Authenticate API requests using Personal Access Tokens. All API requests must include your access token in the Authorization header.

POST /v1/auth/authenticate Authenticate with username and password POST /v1/auth/refresh Refresh an access token POST /v1/auth/revoke Revoke an access token

Accounts

Create and manage developer accounts. Each account can generate multiple Personal Access Tokens for API authentication.

POST /v1/accounts Create a new developer account GET /v1/accounts/{account_id} Get account details PATCH /v1/accounts/{account_id} Update account information

Access Tokens

Generate and manage Personal Access Tokens for API authentication. Tokens are scoped to your account and can be revoked at any time.

POST /v1/access-tokens Create a new Personal Access Token GET /v1/access-tokens List all access tokens for your account DELETE /v1/access-tokens/{token_id} Revoke an access token

Applications

Create and manage applications. Each application automatically gets an Ethereum wallet and can have multiple API keys. Applications can be in sandbox (testnet) or production (mainnet) mode.

POST /v1/applications Create a new application (auto-creates wallet) GET /v1/applications List all applications GET /v1/applications/{application_id} Get application details PATCH /v1/applications/{application_id} Update application settings DELETE /v1/applications/{application_id} Delete an application

Wallets

Manage Ethereum wallets for your applications. Each application can have multiple wallets. Wallets are automatically created when you create an application, but you can create additional wallets as needed.

POST /v1/applications/{application_id}/wallets Create a new wallet for an application GET /v1/applications/{application_id}/wallets List all wallets for an application GET /v1/applications/{application_id}/wallets/{wallet_id} Get wallet details DELETE /v1/applications/{application_id}/wallets/{wallet_id} Archive a wallet

Application Tokens

Create API keys scoped to specific applications. These tokens inherit the application's environment (sandbox/production) and can be used to make authenticated API calls.

POST /v1/applications/{application_id}/tokens Create a new API key for an application GET /v1/applications/{application_id}/tokens List all API keys for an application DELETE /v1/applications/{application_id}/tokens/{token_id} Revoke an API key

Webhooks

Receive real-time notifications for deposits, withdrawals, yield accruals, and rate changes. Powered by Svix with automatic retries and signature verification.

GET /v1/webhooks/event-types List available event types POST /v1/webhooks Create a webhook endpoint GET /v1/webhooks List webhook endpoints GET /v1/webhooks/{id} Get webhook endpoint details DELETE /v1/webhooks/{id} Delete a webhook endpoint POST /v1/webhooks/{id}/test Send a test webhook event

Webhook Events

EVENT deposit.completed Funds successfully deposited to a yield account EVENT withdrawal.completed Funds successfully withdrawn from a yield account EVENT yield.accrued Yield accrued to account (triggered daily) EVENT rate.changed Yield rate changed significantly (>0.5%) EVENT account.status.changed Yield account status changed EVENT application.created New application created in your account EVENT api_key.created New API key generated for an application

Yield

Create yield accounts, manage positions, and track yield generation across supported cryptocurrencies. TBD wraps Morpho and Aave protocols to provide a unified 6% yield interface.

Yield Accounts

Create and manage yield accounts for earning interest on cryptocurrency deposits. Each account is backed by Morpho or Aave protocols.

POST /v1/yield/accounts Create a new yield account GET /v1/yield/accounts List all yield accounts GET /v1/yield/accounts/{account_id} Get yield account details POST /v1/yield/accounts/{account_id}/deposit Deposit funds into a yield account POST /v1/yield/accounts/{account_id}/withdraw Withdraw funds from a yield account

Positions

View and manage your active yield positions across different protocols and cryptocurrencies.

GET /v1/yield/positions List all yield positions GET /v1/yield/positions/{position_id} Get position details

Yield Rates

Get current and historical yield rates for supported cryptocurrencies and protocols.

GET /v1/yield/rates Get current yield rates GET /v1/yield/rates/history Get historical yield rates

Yield History

Retrieve historical yield accrual data for your accounts and positions.

GET /v1/yield/history Get yield accrual history

Markets

Explore available markets and protocols. TBD aggregates data from Morpho and Aave to show you all available yield opportunities.

Markets

List all available markets across Morpho and Aave protocols with current rates and availability.

GET /v1/markets List all available markets GET /v1/markets/{market_id} Get market details

Transactions

Track all transactions including deposits, withdrawals, and yield accruals across your accounts.

Transactions

List and retrieve transaction details for deposits, withdrawals, and yield accruals.

GET /v1/transactions List all transactions GET /v1/transactions/{transaction_id} Get transaction details
