Getting Started

Get up and running with Ground APIs in minutes. Create an account, generate an access token, and make your first API call.

Quick Start

  • 1. Create Account
  • 2. Generate Token
  • 3. Make API Call
  • 4. Next Steps
1

Create Your Account

Start by creating a TBD developer account. You'll need a username and password.

Terminal 
curl https://api.ground.com/v1/accounts \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your_username",
    "password": "your_secure_password",
    "email": "your_email@example.com"
  }'
Response 
{
  "account_id": "acc_1234567890abcdef",
  "username": "your_username",
  "created_at": "2025-01-15T10:30:00Z",
  "status": "active"
}

Note: You can also create an account through our web interface.

2

Generate a Personal Access Token

Authenticate with your account credentials to receive an access token, then create a Personal Access Token for API calls.

Step 2a: Authenticate

Terminal 
curl https://api.ground.com/v1/auth/authenticate \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{
    "username": "your_username",
    "password": "your_secure_password"
  }'
Response 
{
  "access_token": "YOUR_ACCESS_TOKEN",
  "token_type": "Bearer",
  "expires_in": 3600,
  "account_id": "acc_1234567890abcdef"
}

Step 2b: Create Personal Access Token

Terminal 
curl https://api.ground.com/v1/access-tokens \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My API Key",
    "expires_in": 31536000
  }'
Response 
{
  "token_id": "tok_1234567890abcdef",
  "access_token": "YOUR_API_KEY",
  "name": "My API Key",
  "created_at": "2025-01-15T10:30:00Z",
  "expires_at": null
}
Important: Save your access token securely. You won't be able to see it again after creation. Use this token in the Authorization: Bearer header for all API requests.
3

Make Your First API Call

Now that you have an access token, let's create your first yield account and start earning 6% yield.

Terminal 
curl https://api.ground.com/v1/yield/accounts \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "currency": "USDC",
    "initial_deposit": {
      "amount": "1000.00",
      "currency": "USDC"
    },
    "protocol_preference": "auto"
  }'
Response 
{
  "account_id": "ya_1234567890abcdef",
  "currency": "USDC",
  "protocol": "morpho",
  "annual_yield_rate": 0.06,
  "status": "active",
  "created_at": "2025-01-15T10:30:00Z",
  "balance": {
    "amount": "1000.00",
    "currency": "USDC"
  }
}

Success! You've created your first yield account. Your funds are now earning 6% annual yield through TBD.s integration with Morpho and Aave protocols.

Next Steps

Now that you're set up, here's what you can do next:

Explore the API

Browse our complete API Reference to see all available endpoints for managing yield accounts, positions, and transactions.

Manage Positions

Use the Positions API to view your active yield positions, check balances, and track yield accruals in real-time.

View Markets

Explore available markets across Morpho and Aave protocols to see current rates and opportunities.

Set Up Webhooks

Subscribe to webhooks to receive real-time notifications for yield accruals, deposits, withdrawals, and more.

Authentication

All API requests must include your Personal Access Token in the Authorization header:

Authorization: Bearer YOUR_API_KEY

Base URL

All API requests should be made to:

https://api.ground.com

API Versioning

Ground APIs are versioned. The current version is 2025-01-15. Include the version in your API path:

https://api.ground.com/v1/yield/accounts
