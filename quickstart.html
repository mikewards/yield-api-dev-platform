Quick Start

Get from zero to your first API call in under 2 minutes. This guide walks you through authentication and making your first request to the TBD API.

Prerequisites

You'll need:

  • A terminal with curl installed (macOS/Linux have this by default)
  • A TBD account (free to create)

1 Create an Account Done

Sign up at tbd.com/signin. No credit card required.

2 Create an Application Done

  1. Go to Dashboard → Applications
  2. Click Create Application
  3. Enter a name (e.g., "My First App")

Your application is auto-configured with Ethereum RPC endpoints for both sandbox (Sepolia testnet) and production (mainnet).

3 Get an API Token Done

  1. Click on your application to open it
  2. Under API Keys, click Create API Key
  3. Select Sandbox environment and click Create
  4. Copy the token (starts with tbd_sand_)

Save this token securely. You won't be able to see it again.

4 Make Your First Request

Run this command in your terminal. Replace YOUR_API_TOKEN with the token you just created.

Terminal 
curl https://flow-platform-flow-platform-staging.up.railway.app/v1/yield/rates \
  -H "Authorization: Bearer YOUR_API_TOKEN"

Response

{ "rates": [ { "currency": "USDC", "protocol": "aave", "network": "ethereum_sepolia", "apy": 0.0523, "updated_at": "2025-12-01T12:00:00Z" }, { "currency": "DAI", "protocol": "aave", "network": "ethereum_sepolia", "apy": 0.0302, "updated_at": "2025-12-01T12:00:00Z" } ] }
