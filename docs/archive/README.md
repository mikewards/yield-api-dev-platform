# Archived Documentation

This directory contains historical documentation that has been superseded by newer documentation or is no longer relevant.

**Note**: These files are preserved for historical reference only. For current documentation, see the main [docs/README.md](../README.md).

## Why Archive?

Documentation is archived instead of deleted to:
- Preserve historical context
- Allow reference to old approaches
- Prevent accidental loss of information

## When to Archive

Move documentation to archive when:
1. It's replaced by newer, more accurate docs
2. The feature/approach it describes is deprecated
3. It's a status update that's no longer current
4. It contains outdated instructions that could confuse users
