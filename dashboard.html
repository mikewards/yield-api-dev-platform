How to get a Sandbox RPC URL

Option 1: Infura (Free)

  1. Go to infura.io and sign up
  2. Create a new project
  3. Select "Sepolia" network
  4. Copy your API key
  5. Use: https://sepolia.infura.io/v3/YOUR_KEY

Option 2: Alchemy (Free)

  1. Go to alchemy.com and sign up
  2. Create a new app
  3. Select "Ethereum" → "Sepolia" network
  4. Copy your API key
  5. Use: https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY
For Ethereum mainnet. Get free API keys from Infura or Alchemy
How to get a Production RPC URL

Option 1: Infura (Free tier available)

  1. Go to infura.io and sign up
  2. Create a new project
  3. Select "Ethereum Mainnet" network
  4. Copy your API key
  5. Use: https://mainnet.infura.io/v3/YOUR_KEY

Option 2: Alchemy (Free tier available)

  1. Go to alchemy.com and sign up
  2. Create a new app
  3. Select "Ethereum" → "Ethereum Mainnet" network
  4. Copy your API key
  5. Use: https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY

💡 Both providers offer free tiers with generous rate limits for development.
