POST

Create Yield Account

Creates a yield account using the provided parameters.

POST /v1/yield/accounts Permissions: YIELD_WRITE 100 req/min

Summary

Create a new yield account to start earning interest on cryptocurrency deposits. TBD automatically routes funds to the best available protocol (Morpho or Aave) to ensure 6% yield.

Request body

Example request

Environment:
Terminal 
curl https://api.tbd.com/v1/yield/accounts \
  -X POST \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "currency": "USDC",
    "initial_deposit": {
      "amount": "1000.00",
      "currency": "USDC"
    },
    "protocol_preference": "auto"
  }'

Response

Response 
{
  "account_id": "ya_1234567890abcdef",
  "currency": "USDC",
  "protocol": "morpho",
  "annual_yield_rate": 0.06,
  "status": "active",
  "created_at": "2025-01-15T10:30:00Z",
  "balance": {
    "amount": "1000.00",
    "currency": "USDC"
  }
}
Error 
{
  "error": {
    "code": "INSUFFICIENT_FUNDS",
    "message": "Insufficient funds to create yield account",
    "type": "invalid_request_error"
  }
}
